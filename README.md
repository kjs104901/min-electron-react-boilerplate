# Minimal electron react boilerplate

Electron 5
React 16
Webpack 4
Babel core 7
Babel loader 8


## Install
``` bash
$ git clone https://github.com/kjs104901/min-electron-react-boilerplate.git
$ cd min-electron-react-boilerplate
$ npm install
```

## Develop
``` bash
$ npm start
```
